## Customisation

We recommend using the HtmlPreviewLoader component as provided, but it can be customised using the `theme` or `cssOverrides` props as required.

### Custom theme

The `theme` prop allows you to override specific design tokens for the HtmlPreviewLoader component:

<Canvas layout="padded" of={CustomTheme} />

```tsx
import type { HtmlPreviewLoaderTheme } from '@guardian/stand/HtmlPreviewLoader';
import { HtmlPreviewLoader } from '@guardian/stand/HtmlPreviewLoader';
import { baseColors, baseSizing } from '@guardian/stand';

const customTheme: HtmlPreviewLoaderTheme = {
	color: {
		blue: {
			background: baseColors.coolPurple[700],
			text: baseColors.orange[300],
		},
	},
};

const Component = () => <HtmlPreviewLoader letter="C" theme={customTheme} />;
```

### CSS overrides

The `cssOverrides` prop allows you to pass custom CSS to the HtmlPreviewLoader component, if the theme prop is not sufficient:

<Canvas layout="padded" of={CssOverrides} />

```tsx
import { HtmlPreviewLoader } from '@guardian/stand/HtmlPreviewLoader';
import { baseColors, baseSizing, baseSpacing } from '@guardian/stand';
import { css } from '@emotion/react';

const customStyles = css`
	border: ${baseSizing.size2Rem} solid ${baseColors.red[500]};
	background-color: ${baseColors.orange[700]};
	color: ${baseColors.warmPurple[300]};
	border-radius: ${baseRadius.corner4Px};
`;

const Component = () => (
	<HtmlPreviewLoader letter="O" cssOverrides={customStyles} />
);
```

## Custom Component Build

If you're not using React/Emotion, or `@guardian/stand` is not compatible with your project, you can create a custom HtmlPreviewLoader component using the styles defined in the `HtmlPreviewLoaderTheme` type.

You will however be responsible for any additional functionality on top of the styles, for example image loading, image fallback, accessibility etc.

See "Custom Component" heading under the "HtmlPreviewLoader" component on [codesandbox.io](https://codesandbox.io/p/sandbox/guardian-stand-components-forked-j8cjhy)

**`css`**

You can import the HtmlPreviewLoader styles as CSS from the package, make sure that your build process supports importing CSS from `node_modules`/`package.json`:

<SandboxCss
	componentName={componentName}
	componentHtml={componentHtml}
	componentCss={componentCss}
/>

Use the `componentHtmlPreviewLoader` variable and the `ComponentHtmlPreviewLoader` type to define your custom styles in TypeScript/JavaScript:
