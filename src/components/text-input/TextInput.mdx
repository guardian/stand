## Customisation

We recommend using the TextInput component as provided, but it can be customised using the `theme` or `cssOverrides` props as required.

### Custom theme

The `theme` prop allows you to override specific design tokens for the TextInput component:

```tsx
import type { TextInputTheme } from '@guardian/stand/text-input';
import { TextInput } from '@guardian/stand/text-input';

const customTheme: Partial<TextInputTheme> = {
	// override tokens here
};

const Component = () => <TextInput theme={customTheme}>Hello</TextInput>;
```

### CSS overrides

The `cssOverrides` prop allows you to pass custom CSS to the TextInput component:

```tsx
import { TextInput } from '@guardian/stand/text-input';
import { css } from '@emotion/react';

const customStyles = css\`
	// custom styles here
\`;

const Component = () => <TextInput cssOverrides={customStyles}>Hello</TextInput>;
```
