## Customisation

We recommend using the Menu component as provided, but it can be customised using the `theme` or `cssOverrides` props as required.

### Custom theme

The `theme` prop allows you to override specific design tokens for the Menu component:

```tsx
import type { MenuTheme } from '@guardian/stand/menu';
import { Menu } from '@guardian/stand/menu';

const customTheme: Partial<MenuTheme> = {
	// override tokens here
};

const Component = () => <Menu theme={customTheme}>Hello</Menu>;
```

### CSS overrides

The `cssOverrides` prop allows you to pass custom CSS to the Menu component:

```tsx
import { Menu } from '@guardian/stand/menu';
import { css } from '@emotion/react';

const customStyles = css\`
	// custom styles here
\`;

const Component = () => <Menu cssOverrides={customStyles}>Hello</Menu>;
```
