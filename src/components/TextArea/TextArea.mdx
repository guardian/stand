# TextArea

A text area component built on [React Aria `TextField`](https://react-aria.adobe.com/TextField) component, see their documentation for the API.

## When to use

- Within a form to collect multi line free-text input from the user.

## Peer dependencies

- `@emotion/react`
- `react`
- `react-dom`
- `react-aria-components`
- `typescript`

See the `peerDependencies` section of `package.json` for compatible versions.

See [custom component build](#custom-component-build) for usage without React/Emotion.

## Example usage

<SandboxReact componentName={componentName} componentTsx={componentTsx} />

```tsx
import { TextArea } from '@guardian/stand/TextArea';

/* types, if required */
import type { TextAreaProps, TextAreaTheme } from '@guardian/stand/TextArea';

const Example = () => <TextArea size="sm" label="Description" />;
```

## Props

| Name                      | Type                                | Required | Default | Description                                                                                                                                 |
| ------------------------- | ----------------------------------- | -------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| `size`                    | `'md'` \| `'sm'`                    | No       | `'md'`  | Controls the input typography size.                                                                                                         |
| `label`                   | `string`                            | No       | N/A     | Visible label rendered above the input.                                                                                                     |
| `renderLabel`             | `(label: string) => ReactNode`      | No       | N/A     | Custom render function for the label. Receives the `label` string and returns a ReactNode for visual enhancement                            |
| `description`             | `string`                            | No       | N/A     | Helper text rendered below the label.                                                                                                       |
| `error`                   | `string`                            | No       | N/A     | Validation error message; displayed when `isInvalid` is `true`.                                                                             |
| `isInvalid`               | `boolean`                           | No       | `false` | Marks the field as invalid and applies error border styling.                                                                                |
| `isDisabled`              | `boolean`                           | No       | `false` | Disables interaction and applies disabled styling.                                                                                          |
| `theme`                   | `TextAreaTheme`                     | No       | N/A     | Custom theme overrides.                                                                                                                     |
| `placeholder`             | `string`                            | No       | N/A     | Temporary text that occupies the text area when it is empty.                                                                                |
| `formInputContainerTheme` | `FormInputContainerTheme` (partial) | No       | N/A     | Override Stand tokens for the wrapping label and description container.                                                                     |
| `cssOverrides`            | `SerializedStyles`                  | No       | N/A     | Custom CSS styles.                                                                                                                          |
| `className`               | `string`                            | No       | N/A     | Additional class name(s).                                                                                                                   |
| `...props`                | `ReactAria TextField` props         | No       | N/A     | All other props from `react-aria-components` [`TextField`](https://react-aria.adobe.com/TextField) (e.g. `value`, `onChange`, `autoFocus`). |

## Customisation

We recommend using the TextArea component as provided, but it can be customised using the `theme` or `cssOverrides` props as required.

### Custom theme

The `theme` prop allows you to override specific design tokens for the TextArea component:

```tsx
import type { TextAreaTheme } from '@guardian/stand/TextArea';
import { TextArea } from '@guardian/stand/TextArea';

const customTheme: Partial<TextAreaTheme> = {
	shared: {
		border: `0.0625rem solid ${baseColors.coolPurple[500]}`,
		borderRadius: '0.5rem',
		error: {
			border: `0.0625rem solid ${baseColors.red[500]}`,
		},
	},
};

const Component = () => <TextArea theme={customTheme}>Hello</TextArea>;
```

### CSS overrides

The `cssOverrides` prop allows you to pass custom CSS to the TextArea component:

```tsx
import { TextArea } from '@guardian/stand/TextArea';
import { css } from '@emotion/react';

const customStyles = css\`
	font-style: italic;
\`;

const Component = () => <TextArea cssOverrides={customStyles}>Hello</TextArea>;
```
