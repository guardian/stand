# RadioGroup

A radio button input collection built on [React Aria `RadioGroup`](https://react-aria.adobe.com/RadioGroup) component, see their documentation for the API.

It provides a labelled, accessible collection of radio buttons with support for descriptions, validation errors, two sizes, and a disabled state.

## When to use

- Within a form to collect one input from a set of mutually exclusive options

## Peer dependencies

- `@emotion/react`
- `react`
- `react-dom`
- `react-aria-components`
- `typescript`

See the `peerDependencies` section of `package.json` for compatible versions.

See [custom component build](#custom-component-build) for usage without React/Emotion.

## Example usage

<SandboxReact componentName={componentName} componentTsx={componentTsx} />

```tsx
import { Radio, RadioGroup } from '@guardian/stand/radio-group';

/* types, if required */
import type {
	RadioProps,
	RadioGroupProps,
	RadioGroupTheme,
} from '@guardian/stand/radio-group';

<RadioGroup
	label="Option"
	description="This is a description for the radio button input"
>
	<Radio value="one">Option 1</Radio>
	<Radio value="two">Option 2</Radio>
</RadioGroup>;
```

## Customisation

We recommend using the RadioGroup component as provided, but it can be customised using the `theme` or `cssOverrides` props as required.

### Custom theme

The `theme` prop allows you to override specific design tokens for the RadioGroup component:

```tsx
import type { RadioGroupTheme } from '@guardian/stand/radio-group';
import { Radio, RadioGroup } from '@guardian/stand/radio-group';
import { baseColors } from '@guardian/stand';

const customTheme: Partial<RadioGroupTheme> = {
	// override tokens here
	shared: {
		radio: {
			color: baseColors['warm-purple'][300],
		},

		indicator: {
			border: `2px dashed ${baseColors['warm-purple'][100]}`,
		},
	},
};

const Component = () => (
	<RadioGroup
		theme={customTheme}
		label="Option"
		description="This is a description for the radio button input"
	>
		<Radio value="one">Option 1</Radio>
		<Radio value="two">Option 2</Radio>
	</RadioGroup>
);
```

<Story of={CustomTheme} />

### CSS overrides

The `cssOverrides` prop allows you to pass custom CSS to the RadioGroup component:

```tsx
import { Radio, RadioGroup } from '@guardian/stand/radio-group';
import { css } from '@emotion/react';
import { baseColors } from '@guardian/stand';

const customStyles = css\`
	// custom styles here
	border: 2px solid ${baseColors['cool-purple'][700]};
	background-color: ${baseColors['cool-purple'][900]};
\`;

const Component = () =>
	<RadioGroup
		cssOverrides={customStyles}
		label="Option"
		description="This is a description for the radio button input"
	>
		<Radio value="one">Option 1</Radio>
		<Radio value="two">Option 2</Radio>
	</RadioGroup>;
```
