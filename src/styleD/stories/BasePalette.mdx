# Base Palette

This palette represents the raw values of the colour palette.

## Example usage

```tsx
import { base } from '@guardian/stand';

const headline = css`
	color: ${base.colors.neutral[100]};
	background: ${base.colors['cool-purple'][400]};
`;

/*
.class-name {
  color: #FFFFFF;
  background-color: #7d8ce4;
}
*/
```
