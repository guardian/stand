# Semantic - Typography Presets

This story showcases the semantic typography presets available in the design system.

## Example usage

```tsx
import { css } from '@emotion/react';
import {
	semanticTypography,
	convertTypographyToEmotion,
} from '@guardian/stand';

const headline = css`
	${convertTypographyToEmotion(semanticTypography['title-lg'])};
`;
```
