# Base - Color Palette

This palette represents the raw values of the colour palette.

In most cases, you should not need to use the base colors directly, and should instead use the semantic color tokens provided by the design system.

## Example usage

```tsx
import { baseColors } from '@guardian/stand';

const headline = css`
	color: ${baseColors.neutral[100]};
	background: ${baseColors['cool-purple'][400]};
`;

/*
.class-name {
  color: #FFFFFF;
  background-color: #7d8ce4;
}
*/
```
