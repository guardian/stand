## getting TemplateComponent read to publish

When `TemplateComponent` is ready to be published:

- [] create "src/template-component.ts" to acts as the index file for "src/components/template-component"
- [] export "src/template-component.ts" from the "src/index"
- [] update the `inputs` in "rollup.config.js"
- [] update "package.json" to include the relevant files.
- [] delete this file!
- [] run `npx changeset`

See [CONTRIBUTING](CONTRIBUTING.md) for more details.
